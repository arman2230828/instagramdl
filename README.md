# InstaDL Project

This project has been split into two independent modules:

- **[Frontend](./frontend/README.md)**: A static frontend (HTML/CSS/JS) ready to be deployed on Vercel.
- **[Backend](./backend/README.md)**: A FastAPI Python backend exposing REST APIs, ready to be deployed on AWS EC2 or Render.

Please navigate to the respective directories for specific deployment and development instructions.
